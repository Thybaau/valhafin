# Product Overview

Valhafin is a financial portfolio management web application that connects to multiple investment platforms (Trade Republic, Binance, Bourse Direct) to automatically download transaction history and visualize financial performance through interactive charts and detailed metrics.

## Core Features

- Secure connection to financial accounts via platform-specific scrapers
- Automatic transaction synchronization
- Performance visualization with interactive charts
- Detailed fee tracking and analysis
- Asset price tracking via Yahoo Finance API
- CSV import for manual data entry
- Multi-account portfolio aggregation

## User Experience

- Modern dark theme interface with blue accents
- Responsive design (desktop, tablet, mobile)
- Real-time data updates
- Secure credential storage using AES-256-GCM encryption

## Target Users

Individual investors managing portfolios across multiple investment platforms who need consolidated performance tracking and fee analysis.
